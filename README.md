How to run
============


Run frontend
--------------

- Go to the folder ``bug-tracker-frontend``. Run ``npm install && npm run``.


![alt text](./images/msg920698248-98173.jpg "Title")


Run backend
-------------

![alt text](./images/img.png "Title")

- Go to the folder ``bug-tracker-backend``. Run ``npm install && npm run``.
